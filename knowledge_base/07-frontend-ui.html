Frontend UI & User Experience

Overview

DaFlavors features a modern, professional dark-themed UI built with Tailwind CSS. The interface emphasizes elegance, usability, and consistency across all user roles while maintaining accessibility and responsive design principles.

Design System

Dark Theme Implementation

Primary Background: bg-black (#000000)

Secondary Background: bg-gray-900 (#111827)

Card Backgrounds: bg-gray-900 with border-gray-700

Text Colors: text-white, text-gray-300, text-gray-400

Accent Colors: Blue-based accents for actions and highlights

Status Colors: Green (active), red (suspended), yellow (inactive)

Typography System

Headers: font-light with increased letter-spacing Body Text: Standard weight with proper line-height Spaced Headers: L E T T E R S P A C I N G for elegance Code/Monospace: font-family: monospace for technical content Button Text: font-medium with tracking-wider

Color Coding System

  • Green: Active status, success messages, available items
  • Red: Suspended status, error messages, unavailable items
  • Yellow/Orange: Inactive status, warning messages
  • Blue: Action buttons, links, information highlights
  • Gray: Neutral states, disabled items, secondary text

Responsive Design

Breakpoint System

Mobile (sm): 640px+ - Single column layouts, stack navigation

Tablet (md): 768px+ - Two-column grids, expanded navigation

Desktop (lg): 1024px+ - Multi-column layouts, sidebar navigation

Large (xl): 1280px+ - Full-width utilization, enhanced spacing

Mobile Adaptations

  • Navigation: Collapsible menu systems
  • Cards: Single-column stacking on mobile
  • Forms: Vertical field arrangement
  • Tables: Horizontal scrolling for data tables
  • Modals: Full-screen on small devices

Touch-Friendly Design

  • Button Sizing: Minimum 44px touch targets
  • Spacing: Adequate spacing between interactive elements
  • Gestures: Swipe-friendly card interfaces
  • Forms: Large input fields for easy typing
